#Lvgl graphic library at STM32F103C8T6 
- BluePill board 
- ILI9341 SPI 320x240 display 
- Rotary encoder
- It is necessary to hack STM32F103C8T6 flash size from 64kB to 128kB in STM32F103C8TX_FLASH.ld file
